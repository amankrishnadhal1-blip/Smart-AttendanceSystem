Problem Statement: 

1. Manual attendance takes time and is prone to errors.
2. Paper-based systems require storage and can be damaged.
3. Need for a fast, accurate, and real-time attendance solution.

Objective:

1. To develop a digital attendance system using RFID and Bluetooth.
2. To store attendance data directly in Google Sheets.
3. To reduce manual work and improve accuracy.

Components Used:

1. RFID Reader (RC522)
2. RFID Tags / Cards
3. Arduino Uno
4. HC-05 Bluetooth Module
5. Laptop with Python installed
6. Google Sheets
7. Jumper Wires & Breadboard

Demo Workflow:

1. Scan RFID card.
2. Data sent via Bluetooth to laptop.
3. Python script processes and uploads to Google Sheets.
4. Attendance record is stored instantly.

Advantages:

1. Saves time and effort.
2. Eliminates human error in attendance.
3. Data accessible anytime from anywhere.

Conclusion:

1. Successfully developed an automated attendance system.
2. Improves accuracy and efficiency in attendance tracking.
3. Can be enhanced with face recognition in the future.
